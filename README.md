# atsmile blog

フロントエンドエンジニア atsmile の技術ブログ。

## Tech Stack

- [Astro](https://astro.build/)
- TypeScript
- Tailwind CSS v4
- [microCMS](https://microcms.io/)（記事管理）

## Hosting

- Cloudflare Pages

## Development

```bash
npm install
npm run dev
```

## Environment Variables

`.env` ファイルをルートに作成してください。

```env
PUBLIC_MICROCMS_SERVICE_DOMAIN=your-service-domain
PUBLIC_MICROCMS_API_KEY=your-api-key
```
